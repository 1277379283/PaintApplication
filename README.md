# PaintApplication

演示效果图
![演示](https://github.com/1277379283/PaintApplication/blob/master/GIF.gif)

手写签名 demo
1.笔刷选择器 支持铅笔，钢笔，圆珠笔，毛笔，记号笔，喷枪
2.支持颜色选择器，设置笔刷不透明度
3.橡皮擦控件，支持调整橡皮擦大小、调整橡皮擦样式
4.支持画板 撤销、恢复功能
5.自定义画板背景、保存图片功能


不足：画笔粗细调整 显示效果不是太好（如有好的效果方式，希望可以一起探讨）
